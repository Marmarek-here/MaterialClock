# MaterialClock
MaterialClock - an app designed to use Google's new design - Material 3 Expressive.


### How to use it?
Download the .apk file, install it and you are good to go

### Which OSes are supported?
Android. iOS will NOT be supported in the closest time. We may port it to Linux, but unsure for Windows.

### Which features it has?
Alarms, clock, timers, basically everything you expecy and need
